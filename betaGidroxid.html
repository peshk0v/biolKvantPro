Основания
Растворимые (щелочи)
Основные оксиды щелочных и щелочноземельных металлов (Ca, Sr, Ba)
Na₂O + H₂O → 2NaOH
BaO + H₂O → Ba(OH)₂
Нерастворимые
Основные оксиды других металлов
CrO + H₂O → Cr(OH)₂
CuO + H₂O → Cu(OH)₂
Исключение
NH₄OH
Амфотерные гидроксиды
Образованы амфотерными оксидами
Cr₂O₃ + 3H₂O → 2Cr(OH)₃
ZnO + H₂O → Zn(OH)₂
Кислоты (кислородсодержащие)
CrO₃ + H₂O → H₂CrO₄
SO₃ + H₂O → H₂SO₄
